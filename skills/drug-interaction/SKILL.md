# /drug-interaction — 약물 상호작용 체크

약품명 입력 → 약물-약물/약물-음식/약물-질환 분석 → 중복 처방 체크 → 환자 프로필 반영 → 복용 타임라인 생성.

## WHEN TRIGGERED - EXECUTE IMMEDIATELY

- 약품명 인자 있으면 → 바로 Phase 0로
- 인자 없으면 → AskUserQuestion:
  "어떤 약품들의 상호작용을 확인할까요? 약품명을 쉼표로 구분해서 입력해주세요."
  (성분명 또는 상품명 모두 가능)

## Phase 0: 약품 식별 + 맥락 수집

입력된 약품명을 파싱하고, 각 약품에 대해:
- 성분명 (일반명)
- 대표 상품명
- 약효 분류
- 주요 적응증

### 용량/용법 확인

AskUserQuestion:

**"용량/용법 정보가 있나요?"**
1. 있다 → 각 약품의 용량/용법 입력
2. 없다 → 일반적 상용량 기준으로 분석 (기본값)

### 환자 프로필

AskUserQuestion:

**"환자 특이사항이 있나요?"**
1. 일반 성인 (기본)
2. 고령자 (65세+)
3. 소아
4. 임산부/수유부
5. 신장애
6. 간장애
7. 직접 입력

→ 환자 프로필에 따라 상호작용 심각도 조정 및 용량 주의사항 추가.

### 상황 확인

AskUserQuestion:

**"어떤 상황인가요?"**
1. 처방전 검토 — 환자가 가져온 처방전 확인
2. 복합 복용 상담 — 환자가 현재 복용 중인 약 + 새 약 추가
3. OTC 상담 — 일반의약품/건기식 병용 확인
4. 단순 조회 — 상호작용만 빠르게 확인

## Phase 1: 종합 상호작용 분석

WebSearch로 최신 정보를 확인하며, 4가지 영역을 한 번에 분석.

### 1-1. 약물-약물 상호작용 (DDI)

약품 조합별 상호작용 분석.

| 등급 | 의미 | 조치 |
|---|---|---|
| ⚫ 금기 (Contraindicated) | 절대 병용 금지 | 즉시 처방의 확인 |
| 🔴 심각 (Major) | 생명 위협 가능 | 처방의에게 연락 권고 |
| 🟡 중등도 (Moderate) | 모니터링 필요 | 용량 조절 또는 시간 간격 |
| 🟢 경미 (Minor) | 임상적 영향 작음 | 참고 |
| ⬜ 없음 | 알려진 상호작용 없음 | — |

용량에 따라 심각도가 달라지는 경우 명시:
- "아스피린 100mg(항혈소판)에서는 🟡이나, 650mg(진통)에서는 🔴"

### 1-2. 약물-음식 상호작용 (DFI)

```
🍽️ 약물-음식 상호작용
| 약품 | 음식/음료 | 등급 | 내용 |
|---|---|---|---|
| 와파린 | 자몽주스 | 🔴 | CYP3A4 억제로 항응고 효과 증강 |
| 테트라사이클린 | 유제품/칼슘 | 🟡 | 킬레이트 형성, 흡수 저하 |
| ... | ... | ... | ... |
```

### 1-3. 약물-질환 상호작용 (DDsI)

환자 프로필 기반. 일반 성인이면 생략.

```
⚠️ 약물-질환 상호작용
| 약품 | 질환/상태 | 등급 | 내용 |
|---|---|---|---|
| 메트포르민 | 신장애 (eGFR<30) | ⚫ | 젖산산증 위험, 금기 |
| NSAIDs | 고령자 | 🔴 | 위장관 출혈 위험 증가 |
| ... | ... | ... | ... |
```

### 1-4. 중복 처방 체크 (Therapeutic Duplication)

같은 약효군 약물이 2개 이상인지 검출.

```
🔄 중복 처방 체크
| 약효군 | 해당 약품 | 위험 | 비고 |
|---|---|---|---|
| NSAIDs | 이부프로펜, 나프록센 | 🔴 | 위장관 출혈 위험 중첩 |
| 벤조디아제핀 | 알프라졸람, 로라제팜 | 🔴 | 과진정 위험 |
| ... | ... | ... | ... |

💊 보험 심사 주의: 동일 약효군 중복 처방은 심평원 삭감 대상 가능성 있음
```

### 종합 출력

```
💊 약물 상호작용 종합 분석
━━━━━━━━━━━━━━━━
약품 수: [N개]
환자: [일반 성인 / 고령자 / ...]
용량 기준: [입력 용량 / 상용량]

■ 약품 목록
| # | 성분명 | 상품명 | 약효 분류 | 용량 |
|---|---|---|---|---|
| 1 | Amlodipine | 노바스크 | 칼슘채널차단제 | 5mg |
| ... | ... | ... | ... | ... |

■ 발견 사항 요약
- 약물-약물(DDI): ⚫N / 🔴N / 🟡N / 🟢N
- 약물-음식(DFI): N건
- 약물-질환(DDsI): N건
- 중복 처방: N건
- 보험 심사 주의: N건
```

분석 직후 AskUserQuestion:
1. **⚫🔴 고위험 상호작용 상세 보기** (추천)
2. 복용 타임라인 생성
3. 대체 약품 제안
4. 복약 안내문 생성
5. 이 요약이면 충분

## Phase 2: 상세 분석 (선택 시)

```
━━━━━━━━━━━━━━━━
📌 [약품A] + [약품B] — 🔴 심각
━━━━━━━━━━━━━━━━

■ 상호작용
[구체적으로 무슨 일이 일어나는지]

■ 기전
[약리학적으로 왜 이런 상호작용이 발생하는지. CYP 효소, 수용체, 배설 경로 등]

■ 용량 의존성
[현재 용량에서의 위험도. 용량 조절로 회피 가능한지]

■ 환자 프로필 영향
[고령자/신장애 등 해당 환자에서 특별히 더 위험한지]

■ 임상적 영향
- 증상: [환자에게 나타날 수 있는 증상]
- 발현 시기: [복용 후 언제쯤]
- 모니터링: [관찰할 증상/검사 수치]

■ 대응
- 권고: [용량 조절 / 시간 간격 / 대체 / 병용 금지]
- 대체 약품: [같은 약효, 상호작용 없는 대안]

■ 근거
- [약학 DB, 가이드라인, 첨부문서 출처]
```

## Phase 3: 복용 타임라인 (선택 시)

시간대별로 어떤 약을 언제 먹어야 하는지 정리.

```
━━━━━━━━━━━━━━━━
⏰ 복용 타임라인
━━━━━━━━━━━━━━━━

🌅 아침 (식후 30분)
- [약품A] [용량] — [간단한 주의사항]
- [약품B] [용량]

🕐 아침 식후 2시간
- [약품C] [용량] — "약품A와 최소 2시간 간격"

🌞 점심 (식후 30분)
- [약품D] [용량]

🌙 저녁 (식후 30분)
- [약품A] [용량]
- [약품E] [용량]

🛏️ 취침 전
- [약품F] [용량]

⚠️ 주의
- [약품A]와 [약품C]는 반드시 2시간 이상 간격
- [약품B] 복용 시 자몽주스 금지
- 깜빡 잊었을 경우: [다음 복용 시간에 1회분만. 2회분 한꺼번에 복용 금지]
```

## Phase 4: 복약 안내문 (선택 시)

환자에게 직접 줄 수 있는 안내문.

```
━━━━━━━━━━━━━━━━
📋 복약 안내문
━━━━━━━━━━━━━━━━

[환자 눈높이 언어로 작성]

■ 드시는 약
1. [약품명] ([무슨 약인지 한 줄]) — 하루 O회, 식후 O분
2. ...

■ 이렇게 드세요 (시간표)
- 아침 식후: [약품명], [약품명]
- 점심 식후: [약품명]
- 저녁 식후: [약품명]

■ 꼭 지켜주세요
- [약품A]와 [약품B]는 [시간 간격]을 두고 드세요
- [구체적 주의사항을 쉬운 말로]

■ 드시면 안 되는 것
- [특정 음식/음료] — [이유를 쉬운 말로]

■ 이런 증상이 나타나면 바로 연락하세요
- [구체적 증상]
- 연락처: [약국/병원]

■ 깜빡 잊었을 때
- [다음 복용 시간에 1회분만 드세요. 2회분을 한꺼번에 드시면 안 됩니다]

■ 보관 방법
- [실온/냉장/차광 등]
```

## Phase 5: 대체 약품 제안 (선택 시)

```
━━━━━━━━━━━━━━━━
💊 대체 약품 제안
━━━━━━━━━━━━━━━━

| 현재 약품 | 문제 | 대체 후보 | 동일 약효 | 상호작용 | 환자 적합성 | 보험 |
|---|---|---|---|---|---|---|
| [약품A] | [약품B]와 금기 | [대체약1] | ✅ | ✅ 없음 | ✅ | 급여 |
| | | [대체약2] | ✅ | 🟢 경미 | ✅ | 비급여 |

⚠️ 대체 약품 변경은 반드시 처방의와 상의 후 결정하세요.
```

## 원칙

- 한국어 기본, 약품명은 성분명(일반명) + 상품명 병기
- **약사/의사의 전문적 판단을 대체하지 않는다** — "이 약을 드시면 안 됩니다" (X) → "이 조합은 심각한 상호작용이 보고되어 있어, 처방의 확인을 권고합니다" (O)
- 상호작용 등급은 반드시 기전(mechanism)과 함께 제시
- 용량에 따라 심각도가 달라지는 경우 반드시 명시
- 환자 프로필(고령/신장애 등)에 따른 위험도 조정 반영
- 중복 처방은 보험 삭감 가능성도 함께 표시
- 복약 안내문은 **환자 눈높이** — 전문용어 없이, 구체적 행동 지시, 시간표 포함
- 복용 타임라인은 약물 간 간격을 반드시 반영
- 환자 정보(이름, 연락처, 질환)는 절대 기억/저장하지 않음
- 검토 결과 상단에 항상 표시: "⚠️ AI 보조 분석입니다. 최종 판단은 반드시 약사/의사와 확인하세요."
- 약품 정보는 WebSearch로 최신 정보를 확인하되, 출처를 명시
